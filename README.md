# IP Change Notifier 📣

## Dependencies 🎛️

- SendGrid (Configured sender, API Key & Node Module)
- [IPInfo.io](https://ipinfo.io/signup) (API Key)

## Dev Setup 🚧

1. `git clone git@github.com:dewhurstwill/ip-change-notifier.git`
2. `cd ip-change-notifier`
3. `cp sample.env .env`
4. Update .env values
5. `npm i`
6. `npm run start:dev`
